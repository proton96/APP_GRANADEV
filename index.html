Proyecto social digital Summer of Code

Licencias Granada

Aplicación web para ayudar al contribuyente a entender qué licencias necesita y cómo seguir el trámite oficial.

Concepto

Motivo y Problema

Haz clic para girar y leer el valor social

Problema y Valor Social

La idea nace porque la información municipal está muy repartida. El ciudadano duda si el trámite es online o presencial y qué documentos necesita.

El fin social es reducir dudas y acercar la administración con un lenguaje más claro, guiando hacia la ventanilla correcta.

Diseño: Inicio

Haz clic para detalles

Puerta de entrada

La home permite buscar y distinguir licencias frecuentes desde el primer vistazo.

  • Acceso rápido a trámites.
  • Lenguaje sencillo.
  • Confianza institucional sin parecer la sede oficial.
Diseño: Catálogo

Haz clic para detalles

Fichas de licencia

Permite ver en una sola pantalla qué presentar, modelo normalizado, tasas y a qué fuente oficial acudir.

Se apoya en patrones de servicios públicos: tarjetas, tablas simples y botones claros.

Herramientas extra

Comparador: Enfrenta licencias (ej. obra menor vs mayor) para que el ciudadano decida fácilmente su trámite correcto.

Actualizaciones: Muestra qué requisitos han cambiado recientemente y cuándo se revisaron internamente.

Desarrollo y Legal

Stack y Compromisos Summer of Code

Tecnología de vanguardia y cumplimiento estricto de todas las bases y compromisos del concurso.

🏗️

Angular & Spring Boot

Clic para ver arquitectura

Angular: Frontend para construir una interfaz modular con componentes reutilizables y navegación fluida.

Spring Boot: Backend robusto para exponer API REST, organizar reglas de negocio y administrar las fuentes de datos.

🐳

PostgreSQL & Docker

Clic para ver infraestructura

PostgreSQL / MySQL: Base de datos relacional para gestionar tipos de licencias, requisitos y el registro de cambios.

Docker: Contenerización completa para asegurar un despliegue ágil, sencillo y un entorno homogéneo.

🏛️

Fuentes Oficiales

Clic para ver alcance

Datos reales: La app se alimenta de la sede electrónica, gestión unificada y páginas de orientación municipal.

Objetivo final: Facilitar al ciudadano la preparación exhaustiva antes de derivarlo al trámite de la ventanilla oficial.

📜

Código Libre e Inglés

Clic para ver bases

Open Source: El proyecto final será liberado y publicado íntegramente como código abierto bajo la licencia MIT en GitHub.

Inglés técnico: Todo el código fuente, comentarios y documentación técnica estarán redactados en inglés.

💡

¿En qué ayudaría esta app?

Clic para ver la solución

Nace de la necesidad real del ciudadano frente a los problemas que encuentra al intentar conseguir una licencia municipal.

Facilitamos todo el proceso proporcionándole exactamente lo que necesita (documentos, modelos, pasos directos) para conseguirla sin confusiones ni pérdidas de tiempo.
